# 🎬 Netflix-Data-Analysis-Dashboard (Interactive Dashboard using MS Excel)

## Project Objective

The objective of this project is to analyze Netflix content data and create an interactive dashboard to understand content trends, genres, ratings, countries, and overall streaming insights using Microsoft Excel.

---

## Dataset Used

- <a href="https://github.com/rutvsoni1105-wbu/Netflix_Data_Analysis_Dashboard/blob/main/netflix_dataset.csv">Netflix Dataset</a>

---

## Questions (KPIs)

* What is the total number of Netflix titles?
* How many Movies and TV Shows are available on Netflix?
* Which genres are most popular on Netflix?
* Which countries have the highest Netflix content?
* How has Netflix content grown over the years?
* What are the most common content ratings?
* How does the Movies vs TV Shows distribution compare?
* What insights can be generated from Netflix content trends?

---

## Process

* Imported and transformed data using Power Query.
* Performed data cleaning and preprocessing.
* Split and cleaned country and genre columns.
* Built data model using Power Pivot.
* Created Pivot Tables and Pivot Charts.
* Designed KPI cards and interactive dashboard layout.
* Developed charts and visualizations for analysis.
* Added slicers for dynamic filtering.
* Applied Netflix-themed dashboard formatting and styling.

## Project Insights

* Movies dominate Netflix content compared to TV Shows.
* Certain genres contribute significantly to total content.
* The United States has the highest number of Netflix titles.
* Netflix content increased rapidly after 2015.
* TV-MA and TV-14 are among the most common content ratings.
* Interactive slicers help analyze trends dynamically.

---

## Final Conclusion

This Netflix Data Analysis Dashboard provides meaningful insights into Netflix content trends and distribution. The dashboard helps understand viewer-oriented content patterns, genre popularity, ratings distribution, and global content availability through interactive visualizations and data analysis techniques.
